<---- Grid ---->

There are many properties that comes with grid

i. Basic properties

"display:grid" to create grid and to make columns and rows using.
"grid-template-columns/rows" to define the number of columns and rows,
template can be written as "repeat(number of times repeated, pattern of columns/rows in fr or other units)".
"fr" is used to make columns/rows in fraction of available space.
"grid-auto-rows/columns" are used to define the size of rows and columns.
"grid-gap" is used to define the gap between columns and rows.
